# 🩺 Diabetes Prediction System

## 📌 Project Overview

The Diabetes Prediction System is a Machine Learning project that predicts whether a patient is likely to have diabetes based on medical information. The model is trained using the Pima Indians Diabetes Dataset and applies Logistic Regression for binary classification.

The project demonstrates the complete machine learning workflow, including data preprocessing, feature selection, model training, prediction, and performance evaluation.

---

## 🚀 Features

- Data preprocessing and cleaning
- Feature selection for improved model performance
- Binary classification using Logistic Regression
- Prediction of diabetes risk for new patient data
- Model evaluation using multiple performance metrics

## 📊 Dataset

This project uses the **Pima Indians Diabetes Dataset**, which contains medical records of female patients, including:

- Pregnancies
- Glucose
- Blood Pressure
- Skin Thickness
- Insulin
- BMI
- Diabetes Pedigree Function
- Age
- Outcome (Target)

You can download the dataset from Kaggle:

https://www.kaggle.com/datasets/uciml/pima-indians-diabetes-database

After downloading, place the file `diabetes.csv` in the project folder before running the program.

## ▶️ Installation

Clone the repository

```bash
git clone https://github.com/TalalKhanTareen/Diabetes-Prediction.git
```

Navigate to the project folder

```bash
cd Diabetes-Prediction
```

Install the required libraries

```bash
pip install -r requirements.txt
```

Run the project

```bash
python diabetes_prediction.py
```
